## 四、发布步骤

### Step 1: 生成本地脱敏包

```bash
bash 日常运维/版本发布.sh
```

### Step 2: 手动脱敏处理

处理 `main.py` 中的敏感内容：
- 删除或注释掉第 880-900 行的 API 文档 JSON 块（含真实域名）
- 替换示例中的 `your-server.example.com` 为 `your-server.com`
- 替换 `noah` 为 `your-username`

### Step 3: GitHub 推送

```bash
cd /tmp/mnemosyne-publish
git init
git remote add origin https://github.com/gymaira1990-jpg/noah-gen3-type2.git
# 注意: 只更新 02-Mnemosyne-记忆宫殿/ 目录
# 不要动其他模块（01-抽屉级联/03-自主进化）
```

### Step 4: Gitee 镜像创建

```bash
# 在 gitee.com/gcat 创建新仓库 noah-gen3-type2
# 从本地推送（不自动从GitHub同步）
cd /tmp/mnemosyne-publish
git remote add gitee https://gitee.com/gcat/noah-gen3-type2.git
git push gitee main
```
