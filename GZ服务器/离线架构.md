## 服务管理

```bash
# 查看状态
sudo -S -p '' systemctl status qwen3-fallback

# 重启
sudo -S -p '' systemctl restart qwen3-fallback

# 日志
sudo -S -p '' journalctl -u qwen3-fallback -n 50 --no-pager
```
