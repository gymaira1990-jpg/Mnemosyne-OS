## 存储状态

- PostgreSQL 记忆数: 可通过 `GET /api/v1/tmt/tree/{user_id}` 查看
- TMT 树结构: L1(memories) → L2(sessions) → L3(daily) → L4(weekly) → L5(profiles)

## SSH 密钥

```bash
# PEM 文件位置
/path/to/your-key.pem   # 大陆服务器
/path/to/your-hk-key.pem   # 香港服务器

# 从保险柜更新
cp "/path/to/your/backup/GCAT的办公室/重要保险柜/腾讯云服务/大陆地区服务器SSH.pem" \
  /path/to/your-key.pem
chmod 600 /path/to/your-key.pem
```

## 已知问题

1. **fail2ban**: 累计封禁大量 IP，新 SSH 连接可能被拒（10分钟自动解封）
2. **OOM 风险**: 9.5G/15G 使用率
3. **Reranker**: GZ 旧版 llama-server 不兼容新版 Qwen3-Reranker GGUF，需升级二进制
