## 三、发布策略

### 保留 GitHub 已有内容
- 文档体系（技术说明书/学习指南/部署指南/接入指南）质量高
- Reflector 反思引擎代码可以合并

### 新增到 GitHub
- 新开发的源码（main.py 最新版 + Provider 代码）
- 新功能文档（产品说明书/CHANGELOG/路线图）
- 升级方案文档

### 版本号对齐
```
GitHub 当前: v3.0 (2026-05-22) ← 稳定版
本地当前:   v2.5 (2026-06-04) ← 开发版

发布时: 本地 v2.5 → GitHub 作为 v2.5-release
        或对齐为 v3.1 (在原来 v3.0 基础上增量)
```
